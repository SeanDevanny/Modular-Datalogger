L5X/XML Online Datalogger

The function of this webpage is to extract data, rung comments and then graph data

from Allen-Bradley PLC's custom Datalogger AOI

Drag and drop your exported L5X file in the Input area below then click the upload button


When you're finished here, you may refer to the home page here or via the navbar at the top of the screen


The pie chart displays sequence steps and how much of the cycle time they take up proportionately

In other words, if you wish to take time off of a cycle, start with the steps that take up the largest area of the pie chart


The datalogger takes an L5X PLC file that's running the datalogger AOI and extracts the data along with related comments.

Sometimes these comments don't appear. If this happens with your file, you'll need to append the file


How to append your file

You must first get the actual tag name of the value you wish to monitor for the tags

The commentless tag is likely aliased when it was fed to the datalogger AOI

You should go to your PLC code and use the tag-name the datalogger provides you upon initial upload to find the actual tag name of the function monitored

Trace the aliased tag back to the source to get the actual tag name


Now you have the actual tag name you're going to want to append the document

Before proceeding, ensure the file input box reads something like "Uploaded file is FileName.L5X"


Populate the first input box with the old tag name (i.e. the one that was displaying without comments)

Then populate the second input box with the tag it was aliased as and hit the "append" button

Make sure you use the full tag name. This will not work if only a part of either tag is used
